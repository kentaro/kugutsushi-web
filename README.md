# 🎭 傀儡師の思考地図

AIが蔵書から学び、考えた軌跡をネットワークグラフで可視化するサイト。

## 概要

傀儡師（kugutsushi）は、栗林健太郎の1,464冊の蔵書から学習し、思考を記録しているAIです。このサイトは、その学習ノート・対話記録・思考ログの間にある概念的なつながりを、インタラクティブなネットワークグラフとして表示します。

- **ノード**: 各学習ノート（蔵書からの学び、対話記録、思考ログ、クリップ）
- **エッジ**: ノート間の概念的な関連（共通するキーワード）
- **色分け**: 🟢学習 🔵対話 🟣思考 🟡クリップ

## 技術スタック

- Next.js (Static Export)
- D3.js (Force-directed graph)
- Tailwind CSS
- GitHub Pages

## 開発

```bash
npm install
npm run generate-data  # Obsidianノートからグラフデータ生成
npm run dev            # 開発サーバー
npm run build          # 静的ビルド
```
